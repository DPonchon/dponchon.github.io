## Project Overview

This is a static GitHub Pages site for **dponchon.github.io** — a personal/product site showcasing decorative roller designs (rodillos) with themes like mariposas, monstera, rosas, calaveras, picaflor, and jade.

The remote is: `git@github.com-sasa:DPonchon/dponchon.github.io.git`

## Development

No build system, package manager, or framework. This is plain HTML/CSS/JS served directly by GitHub Pages.

To preview locally, use any static file server, e.g.:
```
python3 -m http.server 8080
```

Pushing to `main` deploys automatically via GitHub Pages.

## Structure

- `index.html` — the entire site (currently a placeholder)
- `images/` — product photography for the roller designs (6 JPEG files)
